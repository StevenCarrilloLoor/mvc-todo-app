Tutorial TODO App MVC con Node.js y Express

En este video se explica cómo crear una aplicación web siguiendo el patrón de arquitectura MVC (Modelo-Vista-Controlador) utilizando Node.js, Express y EJS para las vistas. Se incluye conexión a base de datos local con MongoDB y despliegue en la nube.
